# 🎉 Summer Party Wheel

Aplicação web interativa para girar uma roleta com prémios durante a festa de verão da empresa.

## ✨ Características

- 🎡 Roleta animada com 19 colaboradores
- 🎁 Dois prémios: **FICA** e **RUA**
- 📊 Rastreamento de participantes já girados
- 📱 Responsivo para dispositivos móveis
- 🎨 Design colorido e atrativo

## 🚀 Como usar

1. Abra o ficheiro `index.html` num navegador web
2. Clique no botão **GIRAR ROLETA**
3. Aguarde pela animação e veja o resultado
4. Clique em **REINICIAR** para começar de novo

## 📋 Como partilhar via Teams

Crie um link para a página hospedada (ex: GitHub Pages) e partilhe com os colaboradores:

```
https://sandisabel02-create.github.io/octocat/
```

## 🎯 Prémios

- **FICA**: O colaborador fica no chão
- **RUA**: O colaborador vai para a rua

## 📝 Notas

- A aplicação controla quais colaboradores já giraram
- Cada colaborador só pode girar uma vez por sessão
- Ao final, apresenta uma mensagem de conclusão
